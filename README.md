# 岸线 · 27 考研作战手册

面向 2027 考研数学一、英语一与 408 的私人交互式计划网站。当前详细计划覆盖 **2026-07-18 至 2026-11-15**。

在线地址：<https://1850741061.github.io/kaoyan-27-plan/>

## 使用

直接双击 `index.html` 即可使用；也可以在本目录启动静态文件服务器：

```powershell
python -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 主要功能

- 根据当前进度生成 121 天分科安排，并按“补齐基础、强化成网、真题与报名”三阶段推进；
- 点击“今日执行面”的任一任务，可展开目标、完成标准、分步清单与当日笔记；
- 支持切换日期、编辑任务以及勾选完成，数据自动保存在当前浏览器；
- 横向总日历写入每天全部任务：拖动空白处浏览，拖动任务卡可调整日期；
- 手机端点击日历任务后，可通过“编辑日期 / 时间 / 名称”灵活调整；
- 每日 17:00–19:00 固定运动，学习任务不能覆盖该时段；
- 含报名资料核对、预报名、正式报名和网上确认提醒；相关日期均标注为参考窗口，需以研招网当年公告为准；
- 包含专注计时、周表、分科进度、七日统计、周复盘和 JSON 数据备份。
- 使用 Supabase 邮箱密码登录，在电脑、手机和平板间自动同步；离线修改先保存在本机，联网后补传；
- 云端采用版本号条件更新，两个设备同时改动时会要求选择保留云端或本机，不会静默覆盖。

## 首次启用云同步

网站只提供登录，不开放注册。请先进入 Supabase 的 `27plan` 项目：

1. 在 **Authentication → Users** 中添加唯一用户，并设置自己的邮箱和密码；
2. 建议在 **Authentication → Providers → Email** 中关闭新用户注册；
3. 打开网站并登录。首次登录会把当前设备的本机计划上传到云端，之后其他设备登录会读取同一份数据。

## 数据与安全

浏览器 `localStorage` 仍作为离线缓存，Supabase 表 `kaoyan_planner_state` 保存云端副本。该表启用了 RLS，只允许登录用户读写与自己 UID 相同的一行。

前端仅包含 Supabase Project URL 与 publishable key；它们必须配合登录 JWT 和 RLS 才能访问数据。不要把 `service_role` 或 secret key 写入前端。更换浏览器或清除站点数据前，仍建议导出 JSON 备份。
